# RoutePlanner Demo Site

Простой статический веб-интерфейс для демонстрации итогового проекта RoutePlanner.

## Запуск

Откройте `index.html` в браузере.

Никаких серверов и внешних зависимостей не требуется. Вся логика расчета находится в `app.js`:

- M/M/1 для задержек в узлах;
- Bellman-Ford;
- Dijkstra;
- проверка SLA;
- визуализация маршрута на SVG-графе.

## Демонстрационные сценарии

- `normal`: маршрут A → C → E → F, SLA выполняется;
- `peak`: маршрут A → C → E → F, SLA выполняется, но время близко к лимиту;
- `incident`: маршрут переключается на A → C → D → F, SLA нарушается.

В модели M/M/1 задержка резко растёт, потому что при увеличении входящего потока λ система приближается к пределу пропускной способности μ. В формуле ожидания есть знаменатель μ − λ. Когда λ почти равно μ, этот знаменатель становится очень маленьким, поэтому среднее время ожидания в очереди резко увеличивается. Если λ ≥ μ, очередь становится неустойчивой и в среднем растёт бесконечно.
